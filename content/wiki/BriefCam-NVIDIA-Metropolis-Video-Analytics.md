## Summary
BriefCam has public NVIDIA Metropolis and NVIDIA GPU evidence for video analytics, including retail organized retail crime integration and a named public-safety deployment context. This page is integration and deployment proof, not a named retail-customer case study.

## Detail
The NVIDIA/Loss Prevention Research Council retail crime source describes an AI-based organized retail crime solution built with applications on the [[NVIDIA-Metropolis]] framework and microservices. In that integrated offering, BriefCam contributes vehicle license plate and model recognition, while AiFi contributes shopper-behavior and item-tracking signals and SureView handles physical security management.

BriefCam's NVIDIA GPU press release provides a second proof point: BriefCam joined the NVIDIA Metropolis Software Partner Program and used NVIDIA GPUs to accelerate video processing, deep learning training, and inference. That source names Hartford Police Department as an early beneficiary of running BriefCam on NVIDIA GPUs, showing non-retail deployment evidence for video investigation, traffic pattern analysis, event security, and public safety.

For retail recommendations, use BriefCam when the customer asks for video search, object search, alerting, license plate recognition, cross-camera analytics, asset protection, organized retail crime response, or turning existing surveillance infrastructure into operational intelligence. Be explicit that the strongest retail source is an NVIDIA/LPRC integration scenario rather than a named retailer deployment.
